PERFORMANCE FILES HANDOVER DOCUMENT
=====================================

Overview:
--------
This document explains all business performance CSV files generated for financial modeling and analysis.
Each file contains 5 years (2020-2025) of monthly performance data with realistic market dynamics,
support/resistance levels, and upward trends as requested.

File Structure:
--------------
All CSV files follow the same column structure:
- period: Monthly timestamp (YYYY-MM format)
- regime: Market condition/descriptive phase
- precision_pct: Model accuracy percentage (94-98% range)
- production_costs: Operational costs in USD
- yield_pct: Monthly return percentage (can be negative during drawdowns)
- balance: Cumulative account balance
